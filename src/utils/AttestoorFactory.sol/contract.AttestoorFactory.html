Intuition Beta Contracts Documentation

AttestoorFactory

Git Source

Inherits: Initializable, Ownable2StepUpgradeable

Author: 0xIntuition

A utility contract of the Intuition protocol. It allows for the deployment of Attestoor contracts using the BeaconProxy pattern.

State Variables

ethMultiVault

The EthMultiVault contract address

IEthMultiVault public ethMultiVault;

attestoorBeacon

The address of the UpgradeableBeacon contract, which points to the implementation of the Attestoor contract

address public attestoorBeacon;

count

The count of deployed Attestoor contracts

uint256 public count;
